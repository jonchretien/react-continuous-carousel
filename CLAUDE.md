# react-continuous-carousel

Thin React wrapper (~150 lines) around the vanilla [continuous-carousel](https://github.com/jonchretien/continuous-carousel) library.

## Commands

```bash
npm run build      # tsup → ESM + CJS + .d.ts in dist/
npm run typecheck   # tsc --noEmit (also aliased as "lint")
```
